Growth

Referral Program

Simple, automatic, costs nothing. Run entirely through Shopify's built-in discount code system.

How It Works

Step 1

Client completes their first session. You send them a unique discount code (e.g. FIRSTREPMIKE).

Step 2

They share it with a friend. Friend gets $10 off their first session at checkout.

Step 3

Once the friend books, the original client gets their next session free.

Message to send after every session: "If you know anyone else trying to break into sales, send them your code — [CODE]. They get $10 off and you get your next one free."

Payments

How to Launch on Shopify

Shopify is the cleanest way to accept payments without building anything custom. Start on the Starter plan at $5/month — you don't need a full store, just shareable payment links.

  1. Go to shopify.com and create an account under "First Rep"
  2. Create three products: Intro Call (Free), 30-Minute Session — $50, 3-Session Package — $120
  3. Connect Shopify Payments to accept credit cards — no third-party processor needed
  4. Install Sesami or Appointo (free booking apps) so clients pick a time slot right at checkout
  5. Set up automatic order confirmation emails so clients get a booking confirmation the moment they pay
  6. For referral codes: go to Discounts → Create Discount. Set $10 off per unique client code. Shopify logs every use automatically.
  7. Share your product links directly on LinkedIn and in your Linktree — no website needed to start
